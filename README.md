## API 
- Kakao Maps API
- 공공데이터포털 주차장 open API
- 소셜로그인 API

<br/>

## 주요 기능 
1. 지도에서 근처에 있는 주차장 마커로 확인
2. 주차장 검색
3. 주차장의 상세 정보 확인
4. 주차장 즐겨찾기 및 리뷰 작성
5. 구글, 깃헙, 페이스북 소셜 로그인
